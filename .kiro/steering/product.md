# 제품 개요

EN:TTOKK는 Tauri 2.x로 구축된 데스크톱 생산성 애플리케이션으로, 노트 작성, 캘린더 통합, 프로젝트 관리 기능을 결합한 도구입니다.

## 핵심 기능

- **노트 관리**: TipTap을 사용한 마크다운 지원 리치 텍스트 에디터
- **캘린더 통합**: Google Calendar 동기화 및 일일 노트
- **프로젝트 관리**: 이슈 추적을 위한 Jira 통합
- **채팅 인터페이스**: AI 기반 채팅 기능
- **볼트 시스템**: 파일 기반 노트 구성 및 관리
- **GitHub 통합**: 저장소 활동 및 사이드바 기능

## 대상 플랫폼

Tauri를 통한 네이티브 시스템 통합이 가능한 크로스 플랫폼 데스크톱 애플리케이션 (Windows, macOS, Linux)

## 아키텍처

하이브리드 데스크톱 앱:
- UI를 위한 React 19 프론트엔드
- 시스템 작업을 위한 Rust 백엔드
- API 통합을 위한 Bun 기반 백엔드 서비스
- 데이터 저장을 위한 로컬 파일 시스템
